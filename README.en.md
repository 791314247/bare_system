# bare_system

#### Description
适用于裸机的设备抽象系统，该系统是将裸机开发中常用的外设如gpio、uart、i2c等抽象出来，然后用统一的接口管理，使裸机开发的应用代码可以做到只依赖与bare_system，而不依赖具体的mcu，大大的提升可复用性。
