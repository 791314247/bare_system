# bare_system

#### 介绍
适用于裸机的IO设备模型，该系统是将裸机开发中常用的外设如gpio、uart、pwm、qspi、spi、i2c、sdio等抽象出来，然后用统一的接口管理。开发时只需要将具体用到的外设注册到系统中，然后应用代码通过查找设备、读、写等操作可以做到只依赖与bare_system，而不依赖具体的mcu，大大的提升稳定性、可复用性。
